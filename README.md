# AT.DIGITAL Website Auditor

A lightweight website auditing tool built to analyze basic website security and SEO readiness.

## Features

* HTTPS Security Check
* Domain Validation
* Website Readiness Score
* Modern Dark UI
* Responsive Design
* Instant Audit Results

## Built With

* HTML5
* Tailwind CSS
* JavaScript

## Purpose

This tool was created to help website owners quickly identify basic security and optimization indicators before conducting a full website audit.
